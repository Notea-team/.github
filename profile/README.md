## 🍵 서비스 소개

차 정보 공유의 즐거움과 개인 맞춤형 차 추천을 통한 최적의 차 경험을 제공하는 서비스 Notea 입니다.
---
## ✨ 주요 기능
### 1. 커뮤니TEA
  - 사용자가 TEA에 대한 정보를 교류할 수 있는 정보의 장을 제공합니다.
### 2. Share my notea 
  - 나의 차 경험을 작성하고 기록하고 공유합니다.
### 3. Tasting notea
  - AI가 작성한 테이스팅 노트에 맞춰 나의 차 취향을 찾는 모험을 떠납니다.
